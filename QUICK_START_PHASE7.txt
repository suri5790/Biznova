Step 1: Get Gemini API Key
  → Go to: https://makersuite.google.com/app/apikey
  → Click "Create API Key"
  → Copy the key

Step 2: Add API Key to Backend
  → Open: backend\.env
  → Add: GEMINI_API_KEY=your_key_here
  → Save the file

Step 3: Start Servers
  → Terminal 1: cd backend && npm run dev
  → Terminal 2: cd frontend && npm start

Step 4: Test It!
  → Open: http://localhost:3000
  → Login to your account
  → Click AI Assistant button (bottom right)
  → Try: "Record 2 Pepsi sold for ₹40"
  → Click "Yes" to confirm
  → ✅ Sale added!

  🎯 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. User sends message in natural language
2. AI detects if it's an ACTION or QUESTION
3. If ACTION → Shows confirmation with details
4. User clicks YES or NO
5. If YES → Updates database
6. If NO → Cancels action
7. AI responds with success/cancellation message
